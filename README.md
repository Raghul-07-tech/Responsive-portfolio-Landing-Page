# Responsive Portfolio Landing Page

## Overview
This project is a **responsive portfolio landing page** designed to showcase personal projects and skills. It focuses on **clean design, usability, and mobile responsiveness**, making it easy for visitors to navigate on both desktop and mobile devices.  

The portfolio demonstrates the use of **HTML5 semantic elements**, **CSS for layout and styling**, and **JavaScript for interactive functionality**, such as a mobile-friendly hamburger menu.

---

## Features
- **Responsive Design**: Layout adapts seamlessly to different screen sizes using CSS media queries.
- **Semantic HTML Structure**: Uses `<header>`, `<section>`, and `<footer>` for better accessibility and SEO.
- **Mobile Navigation**: Hamburger menu appears on smaller screens for easy navigation.
- **Project Showcase**: Highlight multiple projects in a clean card layout.
- **Smooth Layout**: Sections for Home, About, Projects, and Contact.

---

## Technologies Used
- **HTML5** – for page structure and semantic markup.
- **CSS3** – for styling, layout, and responsive design using media queries.
- **JavaScript** – for interactive features like the hamburger menu toggle.
